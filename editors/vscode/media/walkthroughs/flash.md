## Flash your board

Flashing installs the Frothy firmware itself. You do this once per board;
after that you only ever send code to the running device.

1. Plug the board in over USB.
2. Run **Frothy: Flash Firmware** and pick your board.
3. Watch the progress in the Frothy terminal.

If the port is reported busy, another program is holding it — run
**Frothy: Stop Serial Sessions** and flash again.
